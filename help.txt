Russian:
Если у вас возникли проблемы с установкой библиотек на Linux, то для установки Python введите:
sudo python3
После этого введите ./pl-pip(linux).sh
Подождите пока установятся все библиотеки. После установки введите python3 PL.py
Если же возникнут другие вопросы, то вы можете написать в Telegramm: Live-knife, также можно обратиться в Discord: live_knife.
Если вам нужно посмотреть другие проблемы, то введите в ElectroBrain: help(site).
Если же у вас Windows или Macos, то установите python на официальном сайте(https://www.python.org/downloads/)
После этого скопируйте код из pl-pip(windows, macos).sh, и введите его в терминал(или же ./pl-pip(windows, macos).sh)
И введите python3 PL.py
English:
If you have problems installing libraries on Linux, then to install Python, enter:
sudo python3
After that type ./pl-pip(linux).sh
Wait until all libraries are installed. After installation type python3 PL.py
If you have any other questions, you can write to Telegram: Live-knife, or you can contact Discord: live_knife.
If you need to view other problems, go to ElectroBrain: help(site).
If you have Windows or Macos, install Python on the official website (https://www.python.org/downloads/).
After that, copy the code from pl-pip(windows, macos).sh and enter it into the terminal (or ./pl-pip(windows, macos).sh)
Enter python3 PL.py
